# parent-gacha

被爸妈的消息整不会了？投个币，帮你接住这一句。粘贴一条家长发来的话，扭蛋机会生成专属「家长人格图鉴」+ 一条能直接发出去的高情商回复。

A gacha machine that helps you reply to Asian parents — turning nagging (and unspoken love) into a shareable card. High-EQ reply generator for messages from Asian parents.

**[→ 在线体验](https://parent-gacha.vercel.app)**

**Tech stack:** Vanilla HTML/CSS/JS · Vercel Serverless Function · Claude API (claude-sonnet-4-6)
